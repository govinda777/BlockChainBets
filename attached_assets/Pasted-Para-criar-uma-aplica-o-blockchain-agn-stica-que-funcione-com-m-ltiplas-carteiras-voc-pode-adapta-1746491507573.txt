# Prompt para Criação de Aplicação Blockchain - OpenLotteryConnect

Estou desenvolvendo uma prova de conceito para meus estudos em blockchain chamada **OpenLotteryConnect** e preciso de sua ajuda para criar uma aplicação funcional e agnóstica em relação a blockchains.

## Sobre o Projeto

OpenLotteryConnect é um protocolo descentralizado que permite que pessoas apostem e criem suas próprias apostas na Web3, funcionando com múltiplas blockchains e carteiras. O sistema deve funcionar como uma plataforma transparente onde:

- **Apostadores** podem participar de diversas modalidades de apostas
- **Casas de apostas** podem criar e gerenciar eventos
- **Especialistas** podem fornecer previsões e análises
- Tudo registrado de forma transparente em qualquer blockchain compatível

## Principais Funcionalidades Desejadas

- Sistema de apostas peer-to-peer sem intermediários
- Smart contracts agnósticos que funcionem em múltiplas blockchains
- Interface amigável e responsiva
- Suporte a múltiplas carteiras (MetaMask, WalletConnect, Phantom, Exodus, etc.)
- Dashboard para acompanhamento de apostas e resultados
- Sistema de reputação para tipsters e bookmakers

## Estrutura de Usuários

O sistema deve contemplar diferentes papéis:
- **Apostadores**: usuários que realizam apostas
- **Casas/Criadores**: entidades que criam eventos e definem odds
- **Especialistas**: fornecem análises e previsões
- **Traders**: realizam operações de compra/venda de apostas

## Referências Técnicas

Gostaria de usar uma stack moderna com:
- Vite + React para o frontend
- Suporte a múltiplas blockchains através de protocolos agnósticos
- Sistema de smart contracts adaptáveis a diferentes redes
- Integração com diversas carteiras (MetaMask, WalletConnect, Coinbase Wallet, Phantom, Exodus)
- Armazenamento descentralizado para histórico de apostas
